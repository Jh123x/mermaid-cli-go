# Mermaid CLI Go

This is a `Golang` CLI that converts a `mermaid` file into a `svg`

## How to use

```bash
go run main.go -i <path to mermaid file>
```

## Current Features

- [x] Works
- [x] Support different content type (`svg`, `png`, `webp`, `pdf`, `md`)
